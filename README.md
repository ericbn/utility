Utility
=======

Adds utility aliases and functions.

Aliases
-------

### General

  | alias | command |
  | ----- | ------- |
  | `chmod` | `chmod --preserve-root -v` |
  | `chown` | `chown --preserve-root -v` |
  | `df` | `df -kh` |
  | `du` | `du -kh` |

### ls

  | alias | command |
  | ----- | ------- |
  | `ls` | `ls --group-directories-first --color=auto` |
  | `l` | `ls -lAh` |
